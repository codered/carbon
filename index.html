Welcome to Carbon.

To use carbon issue the following commands:

$ cd < DIR TO INSTALL > 
$ git clone git@github.com:codered/carbon.git
$ git checkout master

Master branch is the latest and stable build. Dev has daily changes and is bleeding edge. As such it is not recommended for production environemnt.

Requirements

Carbon uses Postgres and has been tested on Ubuntu with Postgres. We recommend you use Ubuntu with RVM for ease of use.

Installing:

$ cd < CARBON DIRECTORY > 
$ bundle install
$ rake db:create
$ rake db:migrate
$ rails s -p 80 -e production
